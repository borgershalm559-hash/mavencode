🤖 Ruflo — Команды для работы

Команды в чате с Claude и в командной строке (CLI)

💬 Команды в чате
💻 CLI команды
Как использовать: Напишите команду прямо в чате (например /sc:implement) или просто опишите задачу словами — я выберу нужный инструмент сам.
⚡ SuperClaude — основные команды
/sc:implement
Разработка
Реализовать фичу или написать код. Активирует нужных агентов (coder, tester, reviewer). Самая используемая команда.
Пример: /sc:implement добавь авторизацию через JWT
/sc:analyze
Анализ
Комплексный анализ кода: качество, безопасность, производительность, архитектура.
Пример: /sc:analyze проверь весь проект на проблемы
/sc:test
Тесты
Запустить тесты, посмотреть покрытие, найти непокрытые места. Генерирует отчёт.
Пример: /sc:test запусти тесты и покажи покрытие
/sc:improve
Улучшение
Улучшить качество кода: рефакторинг, производительность, читаемость.
Пример: /sc:improve оптимизируй этот модуль
/sc:troubleshoot
Отладка
Найти и исправить баги, ошибки сборки, проблемы деплоя.
Пример: /sc:troubleshoot почему падает сборка?
/sc:design
Архитектура
Спроектировать архитектуру системы, API, компоненты. Выдаёт детальные спецификации.
Пример: /sc:design спроектируй микросервис для платежей
/sc:build
Сборка
Собрать проект с умной обработкой ошибок.
Пример: /sc:build
/sc:document
Документация
Написать документацию для компонента, функции или API.
Пример: /sc:document задокументируй этот модуль
/sc:git
Git
Git-операции с умными сообщениями коммитов, PR, ветки.
Пример: /sc:git создай коммит с описанием изменений
/sc:research
Исследование
Глубокое исследование темы через веб-поиск и анализ.
Пример: /sc:research лучшие практики JWT в 2024
/sc:spawn
Оркестрация
Запустить несколько специализированных агентов параллельно для сложной задачи.
Пример: /sc:spawn сделай полный code review проекта
/sc:task
Задачи
Выполнить сложную задачу с управлением воркфлоу и делегированием.
Пример: /sc:task мигрируй базу данных на новую схему
/sc:help
Справка
Показать все доступные /sc команды с описаниями.
Пример: /sc:help
🌐 Ruflo Swarm — мультиагентные команды в чате
/swarm
Рой
Запустить рой агентов на задачу. Агенты работают параллельно и координируют результаты.
Пример: /swarm проведи полный аудит безопасности
/agent
Агент
Запустить конкретного агента: coder, tester, security-auditor, reviewer, planner и др.
Пример: /agent security-auditor проверь этот файл
/memory
Память
Работа с векторной памятью: сохранить факт, найти по смыслу, показать историю.
Пример: /memory search "паттерны аутентификации"
💬 Можно просто написать словами (без slash-команд)
Написать/создать код
Свободный текст
Просто опишите задачу — что написать, что исправить, что улучшить. Я сам выберу нужный инструмент.
Пример: «создай REST API для управления пользователями»
Отладка и ошибки
Свободный текст
Спросите про ошибку или вставьте стек-трейс — я найду причину и исправлю.
Пример: «почему у меня CORS ошибка при запросе к API?»
Использование: npx ruflo@latest <команда> [подкоманда] [опции]
Или если установлен глобально: ruflo <команда>
🚀 Основные команды
КомандаОписание и пример
init
Инициализировать Ruflo в текущем проекте (создаёт .claude/, .mcp.json и т.д.)
npx ruflo@latest init --wizard
status
Показать статус всей системы: daemon, агенты, память, MCP
npx ruflo@latest status
doctor
Проверить здоровье системы и получить список исправлений
npx ruflo@latest doctor --fix
start
Запустить оркестрационную систему Ruflo
npx ruflo@latest start
🌐 Swarm — рой агентов
КомандаОписание и пример
swarm init
Создать рой агентов с нужной топологией
npx ruflo@latest swarm init --topology hierarchical --max-agents 8
swarm status
Статус роя: агенты, задачи, нагрузка
npx ruflo@latest swarm status
swarm "задача"
Запустить рой на выполнение задачи
npx ruflo@latest swarm "сделай code review проекта" --agents 5
🤖 Agent — управление агентами
КомандаОписание и пример
agent spawn
Запустить агента нужного типа
npx ruflo@latest agent spawn -t coder --name my-coder
agent list
Список активных агентов
npx ruflo@latest agent list
agent run
Запустить конкретного агента на задачу
npx ruflo@latest agent run security-auditor
agent status
Статус конкретного агента
npx ruflo@latest agent status <id>
💾 Memory — векторная память
КомандаОписание и пример
memory init
Инициализировать базу векторной памяти
npx ruflo@latest memory init
memory store
Сохранить данные с векторным эмбеддингом
npx ruflo@latest memory store -k "auth-pattern" --value "JWT с refresh токеном"
memory search
Семантический поиск по сохранённым данным
npx ruflo@latest memory search -q "паттерны авторизации"
memory stats
Статистика: количество записей, векторов, неймспейсы
npx ruflo@latest memory stats
🐝 Hive-Mind — коллективный разум
КомандаОписание и пример
hive-mind init
Создать Hive-Mind кластер с Byzantine-консенсусом
npx ruflo@latest hive-mind init --consensus raft
hive-mind spawn
Добавить агента в рой
npx ruflo@latest hive-mind spawn --role worker
hive-mind status
Статус кластера: кворум, лидер, агенты
npx ruflo@latest hive-mind status

security audit

npx ruflo@latest security audit --full

npx ruflo@latest performance profile
performance benchmark

🔄 Workflow — автоматизация
КомандаОписание и пример
workflow create
Создать автоматический воркфлоу из шагов
npx ruflo@latest workflow create --name deploy-pipeline
workflow run
Запустить воркфлоу по имени
npx ruflo@latest workflow run deploy-pipeline
workflow list
Список всех воркфлоу
npx ruflo@latest workflow list
⚙️ Утилиты
КомандаОписание и пример
daemon start
Запустить фоновый daemon (12 воркеров: аудит, оптимизация и т.д.)
npx ruflo@latest daemon start
daemon stop
Остановить фоновый daemon
npx ruflo@latest daemon stop
config set
Установить параметр конфигурации
npx ruflo@latest config set maxAgents 10
autopilot enable
Включить автопилот — агенты будут работать самостоятельно
npx ruflo@latest autopilot enable
analyze
Анализ кода: diff, риски, рецензенты
npx ruflo@latest analyze diff
cleanup
Удалить временные файлы и артефакты Ruflo
npx ruflo@latest cleanup
